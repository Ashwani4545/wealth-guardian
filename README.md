# Wealth Guardian

Wealth Guardian is a modern web application built with Next.js and TypeScript, designed to help users manage and track their financial goals. The project leverages Tailwind CSS for styling and includes a set of reusable UI components for rapid development.

## Features

- User authentication (login & signup pages)
- Responsive navigation bar
- Feature highlights and key benefits
- Modular and reusable UI components (accordion, dialog, card, etc.)
- Theming support
- Clean and modern design with Tailwind CSS

## Project Structure

```
app/
  - Main application pages (home, about, features, login, signup)
components/
  - Reusable UI components and feature cards
hooks/
  - Custom React hooks
lib/
  - Utility functions
public/
  - Static assets
styles/
  - Global styles
```

## Getting Started

### Prerequisites
- Node.js (v18 or higher recommended)
- pnpm (or npm/yarn)

### Installation

1. Clone the repository:
   ```sh
   git clone <repo-url>
   cd wealth-guardian
   ```
2. Install dependencies:
   ```sh
   pnpm install
   # or
   npm install
   # or
   yarn install
   ```
3. Run the development server:
   ```sh
   pnpm dev
   # or
   npm run dev
   # or
   yarn dev
   ```
4. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Scripts
- `dev` - Start the development server
- `build` - Build the application for production
- `start` - Start the production server
